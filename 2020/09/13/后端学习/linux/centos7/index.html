centos7相关命令

centos后台运行jar包

  1. java -jar *.jar

    特点：当前ssh窗口被锁定，可按CTRL + C打断程序运行，或直接关闭窗口，程序退出

  2. java -jar *.jar &

    &代表在后台运行。
    特定：当前ssh窗口不被锁定，但是当窗口关闭时，程序中止运行。

  3. nohup java -jar *.jar &

    nohup 意思是不挂断运行命令,当账户退出或终端关闭时,程序仍然运行

1.查看进程, ps命令

参数：

a ： 显示当前终端下的所有进程信息，包括其他用户的进程

u ：以用户为主的进程状态

x ：通常与 a 这个参数一起使用，显示当前用户在所有终端下的进程信息

-e：显示系统内所有的进程信息

-l ：使用长格式显示进程信息

-f ：使用完整的格式显示进程信息
