百度搜索资源管理平台添加网站

  1. 进入站点管理页面后点击添加网站
  2. 输入网站
  3. 设置站点属性
  4. 验证网站
  • 文件验证：下载文件后放到网站的根目录下即可
  • HTML验证：将指定的<meta>标签放到网站首页(即index.html文件)的<head>标签中

注意：网站验证时不可出现跳转，否则会报302错误。可以把网站部署到80端口上。

hexo设置主动推送

添加网站之后需要向百度推送资源(收录)，一共有三种方式：

  1. API提交(本文用到的方式)
  2. 自动推送(需要关联主体)
  3. sitemap
  4. 手动提交
