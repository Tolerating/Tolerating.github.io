使用Jenkins自动部署Vue项目

新建一个任务,类型为Freestyle project

General中点击高级，勾选使用自定义的工作空间，目录为盘符:\...\nginx\html\myVue(注意设置nginx的配置文件)

源码管理中勾选 Git,然后输入你的仓库地址,选择连接的账户密码。若没有点击添加按钮自行添加。

构建触发器中勾选触发远程构建，输入你自定义的身份验证令牌（webhooks中要用到）

构建环境中勾选Provide Node & npm bin/ folder to PATH,即构建环境为NodeJS
