使用Jenkins自动部署Vue项目

新建一个任务,类型为Freestyle project

1.General

  • 勾选github项目，输入项目仓库的url地址

  • 点击高级勾选使用自定义的工作空间，目录为盘符:\...\nginx\html\myVue(注意设置nginx的配置文件)

2.源码管理

勾选 Git,然后输入你的仓库地址,选择连接的账户密码。若没有点击添加按钮自行添加。

3.构建触发器

勾选触发远程构建输入你自定义的身份验证令牌

（身份令牌在webhooks中要用到,格式在输入框下方，TOKEN_NAME就是你输入的）

只需在Payload URL中输入，其他默认即可

4.构建环境

勾选Provide Node & npm bin/ folder to PATH,即构建环境为NodeJS

我本来是在Jenkins中配置自动安装NodeJS的,但是发现每次构建的时候都会卡在下载NodeJS这里，后来实在没有办法只能选择Jenkins插件中提供的NodeJS
